## 三、用户系统功能

### 3.1 认证模块

#### 登录页面 (/login)
- 用户名输入框
- 密码输入框
- 登录按钮
- 跳转到注册页面链接

- 用户名

- 个人信息
- 我的文章
- 写文章
- 退出登录

- 头像上传功能

#### 文章管理 (/manage/articles)
- 已发布文章列表
- 草稿箱

**功能特性**：
- 文章元数据：标题、Slug、摘要输入

- **编辑操作**：
  - 后退/前进（撤销/重做）
  - 保存
  - 美化排版
- **预览模式**：
  - 浏览器全屏
  - 屏幕全屏
  - 预览
  - 仅预览
  - HTML代码预览
  - 目录展示
- **发布设置**：
  - 上传封面图
  - 保存草稿
  - 发布文章

### 4.4 API接口设计

```
GET  /api/categories              # 获取分类列表
GET  /api/tag/list               # 获取标签列表
GET  /api/chat/messages          # 获取聊天消息
GET  /api/public/articles        # 获取公开文章列表
GET  /api/article/{slug}         # 获取文章详情
POST /api/auth/login             # 用户登录
POST /api/auth/register          # 用户注册
POST /api/article/create         # 创建文章
POST /api/article/update         # 更新文章
POST /api/upload/image           # 图片上传
GET  /api/user/profile           # 获取用户信息
POST /api/user/profile/update    # 更新用户信息
```

## 九、总结

LiliCould.cn 是一个功能完善、设计简洁的个人技术博客。博主"立里可"展示了扎实的技术功底，网站涵盖了从Java后端、Spring Security、数据库到前端Vue3等多个技术领域的内容。

### 网站优势

1. **技术栈现代**：采用Vue3 + Spring Boot主流技术栈
2. **功能完整**：包含博客核心功能 + 实时聊天室特色功能
3. **用户体验好**：响应式设计、代码高亮、目录导航等细节到位
4. **内容质量高**：技术文章系统性强，有深度

### 整体评价

这是一个典型的个人技术博客成功案例，适合作为技术学习和分享的阵地。网站的代码编辑器、实时聊天室等功能体现了博主的开发能力，内容质量也较高，是一个值得关注的技术博客。
